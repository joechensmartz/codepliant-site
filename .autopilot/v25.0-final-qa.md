### Precision Estimate

Based on manual review of detected services against known repo stacks:

- **True positives:** High. All detected services correspond to actual dependencies (Stripe in payment repos, PostHog in analytics-using repos, AI SDKs in AI-enabled repos, Ruby/Rails gems in Ruby projects).
- **False positives:** Low. Some "other" category items (zod, @tanstack/react-query, puma, rack-attack, sidekiq, bullmq) are utility/framework libraries that don't directly process personal data but are legitimately part of the tech stack for compliance documentation.
- **False negatives:** Moderate. Some services visible in configs but not detected via package scanning (e.g., Cloudflare, Vercel infrastructure, specific OAuth providers configured via env vars only).
- **Estimated precision:** ~85% (true positives / (true positives + false positives))
- **Estimated recall:** ~70% (true positives / (true positives + false negatives))
