## 2. HTML Output — NOT IMPLEMENTED

The `--format` flag does not exist in `src/cli.ts`. All format flag variants fail:

| Command | Result |
|---------|--------|
| `--format all` | ERROR: treats `all` as project path |
| `--format markdown` | ERROR: treats `markdown` as project path |
| `--format html` | ERROR: treats `html` as project path |
| No flag (default) | PASS — generates .md files as before |

**Root cause:** The CLI argument parser (lines 99-112 in `src/cli.ts`) has no `--format` case. The `src/output/` directory exists but is empty — no HTML renderer has been created.
