## 6. Known Issues

**Minor:** `codepliant diff` reports HTML/widget files as "would be removed" when the existing `legal/` directory was generated with `--format all` but diff regenerates with the default markdown format. This is technically correct behavior (those files wouldn't be in a markdown-only regeneration) but could confuse users. Consider either: (a) having diff respect the outputFormat from .codepliantrc.json, or (b) only diffing .md files.
